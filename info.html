This sends information about various things.

iab!info emote [-ec]

OPTIONAL: -ec will condense the output to be on one line and not include emote names

iab!info invite [-ic]

OPTIONAL: -ic will use more concise messages for each invite, lowering the total output's size.

iab!info Guild

Sends some info about the server

iab!info general [-ec] [-ic]

Equivalent to iab!info guild + iab!info emote (-ec if specified) + iab!info invite (-ic if specified)

iab!info user %

Shows some info about the targeted user.
