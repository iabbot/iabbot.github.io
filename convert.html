iab!convert

Back to Index

iab!convert submodule inputunit count outputunit

submodule: see submodules.

inputunit: see units under submodule

count: the number of input units

outputunit: see units under submodule


Example:

iab!convert storage yobibyte 4 gigabyte

4.83570327845852E+15 gb


Any submodule that mentions it has metric prefixes means all of the following are accepted:

Yocto (y), Zepto (z), Atto (a), Femto (f), Pico (p), Nano (n), Micro (u), Milli (m), Centi (c), Deci (d), Deka (da), Hecto (h), Kilo (k), Mega (mm), Giga (g), Tera (t), Peta (pp), Exa (e), Zetta (zz), Yotta (yy)
