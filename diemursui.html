iab!die

Usage

iab!die #
# refers to the number of messages to go through, deleting each of iab's messages.

Example

iab!die 100

Out of the last 100 messages, it deletes iab's.


iab!sui

Syntax

iab!sui #

# refers to the number of messages to go through, deleting each of your messages.

Example

iab!sui 100
Out of the last 100 messages, it deletes yours.
It's literally the exact same thing as iab!die internally, but it uses your id instead of iab's id.


iab!mur

Usage

iab!mur % #

% refers to the user whose messages to delete. Check the selector reference for targeting information.
# refers to the number of messages to go through.

Example

iab!mur iabd 100

Out of the last 100 messages, it deletes the targeted user's.

It's literally the exact same thing as iab!die internally, but it uses the target's id instead of iab's id.
