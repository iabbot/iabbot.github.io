What is the -colorSpace option in Filter 2 (Desaturate)?

There are a lot of ways to desaturate a pixel. Not all of them are the 'best' way. For example, the default action for AKAIP to take when you don't specify an option is to take (R + G + B ) / 3 as each channel's new value. Quite frankly, this is a terrible way to do it; it creates a light, washed-out version of the image. As such, I've introduced new options for the command:

0: Old AKAIP method

This method, described before, takes every channel into consideration:

B = (R+G+B) / 3

1: HSV method

This method takes into account which channel is the highest in its calculation.

B = max(R,G,B)

2: HSL method

This method takes both the lowest and highest channel into account.

B = (max(R,G,B) + min(R,G,B)) / 2

3: HSP method

This method takes all channels into account by how the human eye sees lightness from each color.

B = sqrt(0.299R^2 + 0.587G^2 + 0.114B^2)
