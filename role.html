iab!role

This command manages roles, self-assignments, and more.

Subcommands

Assign

Syntax

iab!role assign % $

% refers to the user
$ refers to the role to assign.

Example

iab!role assign うん#6878 Ultimate
This command would assign the role Ultimate to the user うん#6878.


Assignall

Syntax

iab!role assignall $

$ refers to the role to assign.

Example

iab!role assignall Member
This command would assign the role Member to every user. Keep in mind this command will only assign one member a role each second.


Unassign

Syntax

iab!role unassign % $

% refers to the user
$ refers to the role to unassign.

Example

iab!role unassign うん#6878 Ultimate
This command would unassign the role Ultimate from the user うん#6878.


Unassignall

Syntax

iab!role unassignall $

$ refers to the role to unassign.

Example

iab!role unassignall Ultimate

This command would unassign Ultimate from every user. Internally, it deletes the role and recreates it in-place to be fast.


Create

Syntax

iab!role create Optional# "

Optional# refers to the color of the role (currently only decimal. use iab!todec on a hex-string to get the decimal value)
" refers to the name of the role.

Example

iab!role create 16777215 Color_White
This command would create a role named Color_White, with a white color (hex FFFFFF).


Delete

Syntax

iab!role delete $

$ refers to the role to delete.

Example

iab!role delete Members
This command would delete the role named Members.


Empty

Syntax

iab!role empty

This command enumerates how many users are in each role. If it's empty, it sends the role's name.

Selfassignadd

Syntax

iab!role selfassignadd $

$ refers to the role to add to the self-assign pool.

Example

iab!role selfassignadd Color_Blue
This command would allow users to iab!role selfassign Color_Blue.


Selfassignremove

Syntax

iab!role selfassignremove $

$ refers to the role to remove from the self-assign pool.

Example

iab!role selfassignremove Color_Blue
This command would no longer allow users to iab!role selfassign Color_Blue.


Selfassign

Syntax

iab!role selfassign $

$ refers to the role to assign.

Example

iab!role selfassign Color_Blue
This command would assign the role Color_Blue to the user うん#6878.
This command does not require administrator privileges.


Whohas

Syntax

iab!role whohas $

$ refers to the role to query.

Example

iab!role whohas Administrator
This command would return a list of users that have the Administrator role. You cannot target the @everyone role.
