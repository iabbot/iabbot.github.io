This command edgemaps an image.

REQUIRED PARAMETER

Image

Can either be attached or linked. This is the image to be edgemapped.


-alphamode #

Sets the mode of alpha calculation. Valid values:

  1. If the pixel is an edge, it will have an alpha of 255. Otherwise, its alpha will be 0.
  2. If the pixel is an edge, it will use the source pixel's alpha. Otherwise, its alpha will be 0.
  3. If the pixel is an edge, its alpha value will be the contrast between it and other pixels, otherwise 0.

-bias #

The amount of alpha to add or remove from the edgemap.


-color #

Defines the color of the edgemap. Valid values are six-digit hexadecimal numbers.


-multiplier float#

Will multiply the alpha by the given amount. This is processsed BEFORE bias.


-radius #

Defines the radius of pixels to consider when looking for edges.


-tolerance #

Defines the amount that one channel must differ for a pixel to be considered an edge.
