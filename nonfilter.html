iab!filter -type (FilterID) (Parameters)

FilterID: see Filters.

This page exists to catalog functions that you can perform under filters that are technically separate from how iab usually handles filters. These filters are usually faster since iab handles them directly, and does not send jobs to another program.

"Filters:"

RotateRight

RotateLeft

RotateFlip (180)

Flip Horizontal

Flip Vertical

Cut X/Y

Repeat X/Y



RotateRight

Rotates an image 90 degrees right.

Possible IDs:

  • RotateRight
  • RotateR
  • RoR

No Parameters.



RotateLeft

Rotates an image 270 degrees right

Possible IDs

  • RotateLeft
  • RotateL
  • RoL

No parameters.



RotateFlip

Rotates an image 180 degrees left or right, you'll never know which one it is.

Possible IDs:

  • RotateFlip
  • RotateF
  • RoF

No Parameters.



Flip X

Flips an image horizontally.

Possible IDs:

  • FlipHorizontal
  • FlipH
  • FlipX

No parameters



Flip Y

Flips an image vertically.

Possible IDs:

  • FlipVertical
  • FlipV
  • FlipY

No parameters.



Cut X / Y

Cuts out a portion of an image.

Possible IDs:

  • CutX
  • CutY

Parameters

  • (-x if CutX, -y if CutY)

    The position in which to cut.

  • (-w if CutX, -h if CutY)

    The amount to cut out.



Repeat X / Y

Repeats a single line of an image for a specified width.

Possible IDs:

  • RepeatX
  • RepeatY

Parameters

  • (-x if RepeatX, -y if RepeatY)

    The position in which to start repeating.

  • (-w if RepeatX, -h if RepeatY)

    The amount of lines to draw.
