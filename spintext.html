iab!spintext

This command manages spinning text generation.

Renders are exclusive, and two cannot be done at once.


Subcommands

Create

iab!spintext create [#zoom] "text
OPTIONAL: #zoom represents the zoom for the camera. Smaller values = closer. Default 0.66
"text is the text to render. It can contain new lines.


Delete

iab!spintext delete #id
#id is the id of the spintext to delete, as returned by the create subcommand.


Deleteall

iab!spintext deleteall

Deletus.


Busy

iab!spintext busy

Will tell you if Blender is busy rendering.


List

iab!spintext List

Will generate a mosaic of your spintexts.
